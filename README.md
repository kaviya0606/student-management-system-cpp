# Student Management System

## Overview

A Console-Based Student Management System developed in C++ that manages student records using file handling and menu-driven operations.

## Features

* Add Student Records
* Update Student Records
* Delete Student Records
* Display Student Records
* File Handling for Permanent Storage
* Menu-Driven Interface

## Technologies Used

* C++
* File Handling (fstream)
* String Streams (sstream)
* Structures
* Functions

## Project Objective

To develop a console-based application that efficiently manages student records and stores data permanently using file handling techniques.
